安装Ubuntu Restricted Extras. 这样可以安装许多由于法律原因而无法与操作系统绑定的重要软件包，如Adobe公司的Flash与Microsoft的字体。自己安装这些软件是没有任何问题的，但开发者不能默认地包含这些软件。打开一个终端，输入如下命令：
    sudo apt-get update
    sudo apt-get install ubuntu-restricted-extras
